# Code of Conduct

This project adopts the **[Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)** as its code of conduct.

The full text is maintained at the link above; please read it before contributing.

## Reporting

If you observe behaviour in this project's spaces (issues, pull requests, discussions, related ATC.Net channels) that you believe falls outside the Contributor Covenant, please report it privately to the ATC.Net maintainers via [GitHub Issues](https://github.com/atc-net/atc-wpf/issues) (use the "Report" affordance for private contact when the matter is sensitive).

Reports will be reviewed and acted upon by the maintainer team.

## Scope

This code of conduct applies within all project spaces — the GitHub repository (issues, PRs, discussions, wiki, code), and any official ATC.Net communication channel referencing this project.
